## Common Problems and Solutions

**Problem 1**: The application doesnt start

**Solution**: Check if all dependancies is installed correct. Run `npm install` or `pip install -r requirements.txt` depend on what technology your using.

**Problem 2**: Getting error messages

**Solution**: Read the error message carefuly and try too understand what its saying. Most times the solution is obvius if you read it properly.
